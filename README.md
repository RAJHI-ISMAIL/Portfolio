### Personal Information

   - Your name and title
   - About section content
   - Contact information
   - Social media links

### Images
Add your own images to the `images/` folder:
- `profile.jpg` - Your professional headshot (300x300px recommended)
- `about.jpg` - About section image
- `project1.jpg`, `project2.jpg`, `project3.jpg` - Project screenshots

### Colors and Styling
Customize the color scheme in `css/style.css`:
```css
:root {
    --primary-color: #0d6efd;    /* Change primary color */
    --secondary-color: #6c757d;  /* Change secondary color */
    /* Add more custom colors */
}
```

### Projects
Update the projects section in `index.html`:
- Add your project titles and descriptions
- Update technology badges
- Add live demo and GitHub links
